# tej-tool-api-1.1.2 版本更新
日期：2024/04/17

## 此次版本更新的範圍
tej-tool-api 1.1.1 -> 1.1.2 

<br>

### 更新方式
```
!pip install --upgrade tej-tool-api
```

<br>

### 功能重構
- 原先非交易日（t）公告的資料會合併至**後一個交易日（t-1）**，現在修改成合併至**前一個交易日（t+1）**。
   - 因bundle update功能不會修改過去的資料，所以若要能正確顯示合併至前一個交易日（t-1）的資料，請重新ingest。
